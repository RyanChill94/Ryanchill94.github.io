5. 严格模式

严格模式（“use strict”）帮组开发甄别一些常见的问题（或者说 js 中的缺陷），同时避免 JavaScript 的滥用。在 ES5， 严格模式是一个可选项，但是在 ES6，它是许多 ES6 特性所依赖的，因此很多开发者和工具库，比如 babel，自动在文件的顶部添加了“use strict”，默认开启 js 的严格模式来强迫我们写更好的 JS 代码。
